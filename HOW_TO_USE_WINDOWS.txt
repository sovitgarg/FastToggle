============================================
   TOGGLE AUTOMATION - QUICK START GUIDE
   (For Windows Users)
============================================


============================================
FOLDER STRUCTURE
============================================

FastToggle/
├── run.bat              <-- Double-click to toggle
├── check_status.bat     <-- Double-click to check status
├── ToggleExcel.xlsx     <-- Edit this file with your URLs
└── output/              <-- Results saved here


============================================
STEP 1: INSTALL PYTHON (One-time)
============================================

1. Go to: https://www.python.org/downloads/
2. Click "Download Python 3.x.x"
3. Run the installer
4. IMPORTANT: Check "Add Python to PATH" at the bottom!
5. Click "Install Now"


============================================
STEP 2: EDIT ToggleExcel.xlsx
============================================

1. Double-click "ToggleExcel.xlsx" to open it

2. Fill in your data:

   | URL          | Toggle | userid         | password     |
   |--------------|--------|----------------|--------------|
   | https://...  | Yes    | your@email.com | yourpassword |
   | https://...  | No     | your@email.com | yourpassword |

   - "Yes" = toggle this URL
   - "No" = skip this URL

3. Save and close the file (Ctrl+S)


============================================
STEP 3: RUN THE TOGGLE
============================================

1. Double-click "run.bat"
2. First time: Setup runs automatically (2-5 minutes)
3. Browser opens automatically
4. DO NOT touch keyboard/mouse while browser is running!
5. Wait for it to finish
6. Results saved in: output/toggle_results.xlsx


============================================
STEP 4: CHECK STATUS (Optional)
============================================

To verify current toggle states:

1. Double-click "check_status.bat"
2. DO NOT touch keyboard/mouse while browser is running!
3. Results saved in: output/status_report.xlsx


============================================
OUTPUT FILES
============================================

output/toggle_results.xlsx:
- status: success or failed
- toggle_state_before: ON or OFF
- toggle_state_after: ON or OFF
- updated_at: Date and time

output/status_report.xlsx:
- toggle_status: ON or OFF
- checked_at: Date and time


============================================
TROUBLESHOOTING
============================================

"Python is not recognized"
→ Reinstall Python, check "Add to PATH"

First run takes long time
→ Normal! Setup installs browser (2-5 minutes)

Login fails
→ Check userid/password in ToggleExcel.xlsx
