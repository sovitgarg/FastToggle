============================================
   TOGGLE AUTOMATION - QUICK START GUIDE
   (For Windows Users)
============================================


============================================
FOLDER STRUCTURE
============================================

FastToggle/
├── run_A_ON.bat          <-- Set all A toggles to ON
├── run_A_OFF.bat         <-- Set all A toggles to OFF
├── run_B_ON.bat          <-- Set all B toggles to ON
├── run_B_OFF.bat         <-- Set all B toggles to OFF
├── ToggleExcel_A.xlsx    <-- URLs for group A
├── ToggleExcel_B.xlsx    <-- URLs for group B
├── check_status.bat      <-- Check current toggle status
└── output/               <-- Results saved here


============================================
STEP 1: INSTALL PYTHON (One-time)
============================================

1. Go to: https://www.python.org/downloads/
2. Click "Download Python 3.x.x"
3. Run the installer
4. IMPORTANT: Check "Add Python to PATH" at the bottom!
5. Click "Install Now"


============================================
STEP 2: CREATE EXCEL FILES
============================================

Create two Excel files:

1. ToggleExcel_A.xlsx - URLs for group A
2. ToggleExcel_B.xlsx - URLs for group B

Excel format (3 columns only):

   | URL           | userid         | password     |
   |---------------|----------------|--------------|
   | https://...   | your@email.com | yourpassword |
   | https://...   | your@email.com | yourpassword |

NO Toggle column needed!


============================================
STEP 3: RUN THE AUTOMATION
============================================

Just double-click the appropriate batch file:

   run_A_ON.bat   --> Turn ON all toggles in ToggleExcel_A.xlsx
   run_A_OFF.bat  --> Turn OFF all toggles in ToggleExcel_A.xlsx
   run_B_ON.bat   --> Turn ON all toggles in ToggleExcel_B.xlsx
   run_B_OFF.bat  --> Turn OFF all toggles in ToggleExcel_B.xlsx

First time: Setup runs automatically (2-5 minutes)
Browser opens automatically - DO NOT touch keyboard/mouse!


============================================
STEP 4: CHECK STATUS (Optional)
============================================

To verify current toggle states:

1. Double-click "check_status.bat"
2. DO NOT touch keyboard/mouse while browser is running!
3. Results saved in: output/status_report.xlsx


============================================
OUTPUT FILES
============================================

output/toggle_results_A_ON.xlsx   - Results from run_A_ON.bat
output/toggle_results_A_OFF.xlsx  - Results from run_A_OFF.bat
output/toggle_results_B_ON.xlsx   - Results from run_B_ON.bat
output/toggle_results_B_OFF.xlsx  - Results from run_B_OFF.bat

Each file contains:
- status: success, failed, or skipped
- desired_state: ON or OFF
- toggle_state_before: ON or OFF
- toggle_state_after: ON or OFF
- updated_at: Date and time


============================================
TROUBLESHOOTING
============================================

"Python is not recognized"
-> Reinstall Python, check "Add to PATH"

First run takes long time
-> Normal! Setup installs browser (2-5 minutes)

Login fails
-> Check userid/password in Excel file
